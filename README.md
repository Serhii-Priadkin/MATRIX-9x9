# MATRIX 9x9

This program everyone can find in famous games(like the JEWELS - MATCH 3 PUZZLE).
  
  Rules are very simple!
  
  1) The program are creating matrix 9x9 with random numbers from 0 to 3.
  
  2) If we have 3 same numbers one after another in column or row,then we delete these numbers and moving numbers from top to bottom.  
 
  3) The program are filing new random numbers on the free places.
  
  4) The program are repeating these actions until we have the matrix without repeating a trinity of numbers.
